My custom programming lang specification

  • Only 1D arrays exist, do dimension control yourself
  • Structs, conceptually, are just arrays with keys instead of indices
    • You can't resize arrays or assign new keys to structs after they are initialized
    • Trying to access outside of the array's bounds, or access a struct's key that doesn't exist, will throw a runtime exception
    • Structs aren't typed, meaning you cannot guarantee an arbitrary struct as input will have a certain key (similar to how arrays can be of arbitrary length)

Variables

Declaring a variable:

type:variable_name = value; (* Assigning a value is optional. It will default to whatever is "empty" for that type, ex. 0 for ints *) (* examples *) int:bank_balance = 100; string:name = "Olivia"; float:temperature = 80.65;

Functions

Example of function declarations, definitions, and execution:

(int, int[] -> int):example_function = (foo, bar):{ (* syntactic sugar for declaration *) return foo + bar[0]; } example_function = (int:biz, int[]:baz -> int):{ (* outside of declaration, cannot use syntactic sugar *) return biz + baz[0]; } print (example_function 10 [10, 20, 30]);

Structs

Example of a struct factory:

(-> struct):make_point = { return [ int:x = 0, int:y = 0, (struct, int, int ->):move = (self, dx, dy):{ self[x] += dx; self[y] += dy; } ]; } struct:point = make_point; point[move] point 10 20;
