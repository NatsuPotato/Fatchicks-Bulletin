Guava

Guava is a script that implements OOP. The twist? You can inherit behaviour and attributes from any number of classes!

hello_world.gv _init() => void { print("Hello world!"); } class Fish inherits Animal, Food { eat@Animal(Food food) => void { ... } eat@Food() => void { ... } }
  • The @Class identifier is only necessary in the case of a collision.
  • If you inherit from ClassA and ClassB that each inherit from ClassC, you must implement every function in ClassC yourself (UNLESS neither ClassA nor ClassB override its implementation). Note that you can still use either class's implementation by calling on super@Class.
