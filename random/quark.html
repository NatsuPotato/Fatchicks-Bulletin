Quark

High level Document Object Model processing language.

Element Matching

element my_element = document.body;

match child of my_element {
    "p" {
        child.innerText = "Hello world!";
    }
    "img[src='/cat.png']" {
        child.src = "/dog.png";
    }
    "*" {
        pop child;
    }
}

Templates

Will return the content of the template wrapped in the specified element type.

templateof "article" my_template(string title, string body) {
    <h1>${ title }</h1>
    <p>${ body }</p>
}

document.body push my_template("My Headline", "Lorem ipsum dolor sit amet.");
